## Overview

This project is a responsive **newsletter subscription component** built as part of a devChallenges.io frontend challenge.

The goal was to recreate the provided design as closely as possible while maintaining clean structure, responsive layout, and consistent spacing.

The component includes:
- Headline and description
- Email input field
- Call-to-action button
- Clean and centered layout

---

## What I Learned

- Structuring accessible form elements using semantic HTML
- Styling form inputs and buttons consistently
- Centering layouts using Flexbox
- Managing spacing using `rem` units
- Applying mobile-first design principles
- Improving UI clarity through typography and visual hierarchy

This challenge helped reinforce clean component structuring and responsive layout techniques.
