## ðŸ“± Responsive Design Approach

This project follows a **mobile-first methodology**:

* On **mobile**, articles are stacked vertically.
* On **tablet (768px and above)**:

  * Article cards switch to a horizontal layout.
  * Images increase in size.
  * Spacing between elements is refined for better readability.

This ensures a smooth visual transition between screen sizes.
