## Overview


This project is a **Testimonial Page component** built with **semantic HTML** and **mobile-first CSS**. The layout adapts from mobile to tablet and desktop. Two parallel testimonial cards show a staggered effect on large screens.

### What I Learned

* Structuring semantic content for testimonials
* Mobile-first CSS and responsive design
* Flexbox layout for both vertical and horizontal alignment
* Using `max-width` and `flex:1` for responsive column sizing
* Creating a staggered parallel layout without absolute positioning
* Styling consistent typography, spacing, and shadows
